# Ai_Quiz1
🤖 AI Python Quiz – Beginner Level  This project is an AI-powered Python quiz designed to test beginners after completing the fundamental concepts of Python programming.

📚 Syllabus Covered

* ✅ Variables
* ✅ Data Types (`int`, `float`, `str`, `bool`, `None`)
* ✅ Identifiers & Keywords
* ✅ Input & Output (`input()`, `print()`)
* ✅ Type Conversion & Type Casting
* ✅ Arithmetic, Comparison, Assignment & Logical Operators
* ✅ String Basics
* ✅ String Indexing & Slicing
* ✅ String Functions (`endswith()`, `replace()`, `find()`, `count()`, `capitalize()`)
* ✅ `if`, `elif`, `else`
* ✅ Nested `if` Statements
* ✅ Beginner Practice Programs (Largest Number, Multiple Check, Grading System, etc.)

🎯 Purpose

This quiz evaluates your understanding of Python fundamentals through conceptual questions, output-based questions, debugging challenges, and small coding exercises. It is ideal for students who have completed the beginner level of Python and want to assess their knowledge before moving on to loops and functions.

🚀 Learning Outcomes

After completing this quiz, you should be able to:

* Understand and use Python variables and data types.
* Take user input and display output.
* Apply operators correctly.
* Work confidently with strings.
* Write decision-making programs using conditional statements.
* Solve basic programming problems using Python logic.

Level: Beginner (Python Fundamentals)
